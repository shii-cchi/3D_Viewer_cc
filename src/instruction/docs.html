3DViewer_v2.0

Программа для визуализации каркасной модели в трехмерном пространстве на языке C++ с использованием фреймворка Qt для реализации графического интерфейса

Программа предоставляет возможность:

  • Загружать каркасную модель из файла формата obj (поддержка только списка вершин и поверхностей);
  • Перемещать модель на заданное расстояние относительно осей X, Y, Z;
  • Поворачивать модель на заданный угол относительно своих осей X, Y, Z;
  • Масштабировать модель на заданное значение;
  • Настраивать тип проекции (параллельная и центральная);
  • Настраивать тип (сплошная, пунктирная), цвет и толщину ребер, способ отображения (отсутствует, круг, квадрат), цвет и размер вершин;
  • Выбирать цвет фона;
  • Сохранять настройки между перезапусками программы;
  • Сохранять полученные («отрендеренные») изображения в файл в форматах bmp и jpeg;
  • Записывать небольшие «скринкасты».

Сборка и запуск программы

Для установки и запуска введите команду make install.

Удаление программы

Для удаления программы используйте команду make uninstall.
